Erhebung personenbezogener Daten

Im Rahmen des Checkout-Prozesses werden personenbezogene Daten (Name, E-Mail-Adresse) über ein Formular abgefragt. Es kann ebenfalls eine Datei (Inskriptionsbestätigung im PDF-Format) hochgeladen werden.
Diese Daten sowie die hochgeladene Datei dienen zur Simulation eines Kaufvorgangs und werden weder gespeichert noch weiterverarbeitet (kein Backend vorhanden).
Außerdem steht ein Kontaktformular auf der Webseite zur Verfügung, wobei personenbezogene Daten (E-Mail-Adresse, Nachricht) über ein Formular erhoben werden.
Ebenso dient das Kontaktformular ausschließlich zu Studienzwecken und die eingegebenen Daten werden nicht gespeichert oder weiterverarbeitet.

Local Storage
